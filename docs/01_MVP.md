# 01 — MVP

## Objetivo do MVP 1

Criar o ciclo educativo mínimo:

```text
Receber dinheiro → pagar contas → guardar reserva → investir em renda fixa → avançar mês → ver resultado
```

## Escopo do MVP 1

### Inclui

1. Perfil local simples.
2. Conta corrente.
3. Renda mensal virtual.
4. Despesas fixas mensais.
5. Pagamento de contas.
6. Reserva de emergência.
7. Tesouro Selic simulado.
8. Avançar mês.
9. Estatísticas básicas.
10. Feedback educativo.

### Não inclui

1. Login.
2. Nuvem.
3. Assinatura.
4. Home broker completo.
5. Bolsa em tempo real.
6. Cotações reais.
7. Dados reais do usuário.
8. Comunidade/social.
9. Ranking online.
10. Recomendações reais de investimento.

## Critério de sucesso

O MVP é bem-sucedido se o usuário consegue:

1. entender quanto recebeu;
2. pagar contas;
3. ver quanto sobrou;
4. aplicar parte do dinheiro;
5. avançar um mês;
6. perceber aumento ou redução do patrimônio;
7. receber feedback educativo claro.

## Primeira jornada do usuário

1. Usuário cria perfil local.
2. Escolhe uma renda mensal fictícia.
3. Recebe saldo inicial.
4. Visualiza contas do mês.
5. Paga contas.
6. Escolhe quanto guardar na reserva.
7. Investe em Tesouro Selic simulado.
8. Avança para o próximo mês.
9. Recebe resumo educativo.

## Métrica interna de saúde financeira

O app pode calcular uma nota de 0 a 100 baseada em:

- contas pagas;
- ausência de saldo negativo;
- existência de reserva;
- consistência de investimento;
- diversificação futura;
- controle de gastos;
- evolução patrimonial.
