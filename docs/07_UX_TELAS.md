# 07 — UX e Telas

## Princípio de UX

O usuário deve entender rapidamente:

1. quanto tem;
2. quanto deve;
3. quanto guardou;
4. quanto investiu;
5. o que aconteceu no mês;
6. o que aprendeu.

## Telas do MVP 1

### 1. Onboarding

Objetivo:

- explicar que o app é simulado;
- criar perfil local;
- escolher renda mensal fictícia.

Conteúdo:

- "Este é um jogo educativo."
- "Os valores são virtuais."
- "O objetivo é aprender, não apostar."

### 2. Dashboard

Elementos:

- saldo em conta;
- reserva;
- investimentos;
- patrimônio total;
- contas pendentes;
- botão "Avançar mês".

### 3. Conta corrente

Elementos:

- saldo;
- extrato;
- entradas;
- saídas.

### 4. Contas do mês

Elementos:

- lista de contas;
- valor;
- status;
- botão pagar.

### 5. Reserva de emergência

Elementos:

- valor guardado;
- meta sugerida;
- explicação simples.

### 6. Renda fixa

Elementos:

- Tesouro Selic simulado;
- valor disponível;
- campo de aplicação;
- explicação de liquidez e segurança.

### 7. Resumo do mês

Elementos:

- renda recebida;
- contas pagas;
- valor investido;
- patrimônio final;
- feedback educativo.

## Mensagens educativas

Exemplos:

- "Boa escolha: você pagou suas contas antes de investir."
- "Atenção: investir sem reserva pode ser arriscado."
- "Seu dinheiro guardado rendeu um pouco este mês."
- "Nem todo mês será perfeito. O importante é aprender e melhorar."
