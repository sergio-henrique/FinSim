# Checklist de Segurança para Crianças e Adolescentes

## Privacidade

- [ ] Coleta apenas dados necessários.
- [ ] Não coleta CPF.
- [ ] Não coleta localização.
- [ ] Não coleta contatos.
- [ ] Não coleta dados bancários reais.
- [ ] Não exige foto.

## Produto

- [ ] Não incentiva especulação.
- [ ] Não usa promessa de riqueza.
- [ ] Não recomenda ativos reais.
- [ ] Não usa pressão psicológica.
- [ ] Não compara usuários de forma prejudicial.
- [ ] Não usa ranking online no MVP.

## Linguagem

- [ ] Mensagens são educativas.
- [ ] Erros são explicados sem humilhar.
- [ ] Risco é explicado de forma clara.
- [ ] Simulação é informada explicitamente.

## Monetização futura

- [ ] Sem compras enganosas.
- [ ] Sem pressão por assinatura.
- [ ] Sem paywall que atrapalhe aprendizado básico.
- [ ] Pais/responsáveis devem entender o que está sendo comprado.
