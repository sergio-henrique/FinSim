# Checklist de Release

## Funcional

- [ ] App abre sem erro.
- [ ] Usuário consegue criar perfil.
- [ ] Usuário consegue ver saldo.
- [ ] Usuário consegue pagar contas.
- [ ] Usuário consegue investir.
- [ ] Usuário consegue avançar mês.
- [ ] Resumo mensal aparece corretamente.

## Qualidade

- [ ] Testes passam.
- [ ] Não há crashes conhecidos.
- [ ] Não há dados de teste expostos de forma indevida.
- [ ] Textos foram revisados.
- [ ] App deixa claro que é simulação.

## Segurança

- [ ] Não coleta dados sensíveis.
- [ ] Não usa rede sem necessidade.
- [ ] Não possui ranking online no MVP.
- [ ] Não usa linguagem manipulativa.
