# 02 — Backlog Inicial

## Épico 1 — Estrutura do projeto

- Criar projeto Android Kotlin.
- Configurar Jetpack Compose.
- Configurar arquitetura de pacotes.
- Configurar testes unitários.
- Criar tema visual inicial.
- Criar navegação básica.

## Épico 2 — Conta corrente

### História: Ver saldo

Como usuário, quero ver meu saldo atual para entender quanto dinheiro tenho disponível.

Critérios de aceite:

- Mostrar saldo atual.
- Mostrar entradas.
- Mostrar saídas.
- Não permitir saldo invisível ou inconsistente.

### História: Receber renda mensal

Como usuário, quero receber uma renda virtual todo mês para simular uma mesada ou salário.

Critérios de aceite:

- Ao avançar mês, uma renda é depositada.
- O depósito aparece no extrato.
- O saldo é atualizado.

### História: Pagar contas

Como usuário, quero pagar contas mensais para aprender que despesas vêm antes de investimentos.

Critérios de aceite:

- Contas aparecem todo mês.
- Usuário pode pagar contas se tiver saldo.
- Conta paga não pode ser paga duas vezes.
- Se não houver saldo, mostrar feedback educativo.

## Épico 3 — Reserva de emergência

### História: Guardar reserva

Como usuário, quero separar dinheiro como reserva para me proteger de imprevistos.

Critérios de aceite:

- Usuário pode transferir da conta corrente para reserva.
- Reserva aparece separada do saldo livre.
- App explica por que a reserva é importante.

## Épico 4 — Renda fixa

### História: Investir em Tesouro Selic simulado

Como usuário, quero investir em um título simples e seguro para aprender renda fixa.

Critérios de aceite:

- Usuário informa valor.
- Sistema verifica saldo.
- Aplicação é registrada.
- Valor rende ao avançar o mês.
- Explicação mostra que é uma simulação.

## Épico 5 — Passagem de mês

### História: Avançar mês

Como usuário, quero avançar o tempo para ver consequências das minhas decisões.

Critérios de aceite:

- Renda mensal é depositada.
- Contas do novo mês são geradas.
- Investimentos são atualizados.
- Estatísticas são recalculadas.
- Resumo educativo é exibido.

## Épico 6 — Educação

### História: Receber feedback

Como usuário, quero receber explicações simples sobre minhas decisões.

Critérios de aceite:

- Feedback curto.
- Linguagem adequada para jovens.
- Sem tom punitivo.
- Sem promessa de lucro.
