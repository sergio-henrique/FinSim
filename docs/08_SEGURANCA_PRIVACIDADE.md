# 08 — Segurança, Privacidade e Responsabilidade

## Público jovem

Como o app é destinado a crianças e adolescentes, segurança e responsabilidade devem ser prioridades.

## Princípios

1. Coletar o mínimo possível.
2. Evitar dados sensíveis.
3. Não usar comunicação entre usuários no MVP.
4. Não usar rankings online no MVP.
5. Não estimular comparação social nociva.
6. Não usar linguagem de promessa financeira.
7. Não incentivar especulação.
8. Não usar notificações manipulativas.

## Dados no MVP

Preferir dados locais:

- nome ou apelido;
- faixa etária opcional;
- renda virtual;
- progresso no jogo.

Evitar:

- CPF;
- endereço;
- escola;
- localização;
- contatos;
- telefone;
- dados bancários reais;
- fotos;
- documentos.

## Aviso educativo

O app deve exibir aviso claro:

"Este aplicativo é uma simulação educativa. Nenhuma informação aqui é recomendação de investimento."

## Linguagem proibida

Evitar:

- "fique rico rápido";
- "lucro garantido";
- "melhor ação para comprar";
- "estratégia infalível";
- "ganhe dinheiro real";
- "aposte tudo".

## Linguagem recomendada

Usar:

- "simulação";
- "aprendizado";
- "risco";
- "planejamento";
- "reserva";
- "disciplina";
- "diversificação";
- "longo prazo".
