# /implement-feature

Implemente uma feature já planejada.

Regras:

- Leia o plano aprovado.
- Siga o CLAUDE.md.
- Faça mudanças pequenas.
- Não misture várias features.
- Crie ou atualize testes.
- Atualize documentação relevante.
- Rode testes quando possível.

Ao final, informe:

1. O que foi implementado.
2. Arquivos alterados.
3. Testes criados/executados.
4. Limitações.
5. Próximo passo recomendado.
